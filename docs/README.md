# 城市空间数据分析方法-USDA 库手册

?> 启动于 Mon Oct 17 2022

`USDA`（Urban spatial data analysis method）库是基于[《城市空间数据分析方法》](https://richiebao.github.io/USDA_CH_final/#/)专著，开发构建的Python库。该库主要是面向城乡规划、风景园林和建筑及生态规划等领域，方便数据处理和分析，探索研究城市相关问题。面向科研和教学，试图推动行业的数字化发展。
